Linear Inequalities

Simple Inequalities:

1. Solve for x: 2x + 5 < 15

Answer: x < 5

2. Determine the values of y: (1/3)y - 7 ≥ 5

Answer: y ≥ 36

3. Solve the inequality: 4x - 8 ≤ 20

Answer: x ≤ 7

4. Find a: 3a + 9 > 24

Answer: a > 5

Harder Inequalities:

5. Solve for x: 2(3x - 4) ≥ 5x + 1

Answer: x ≥ 11/7

6. Determine b: (1/2)(4b + 3) - 5 ≤ 2b + 4

Answer: b ≤ 17/6

7. Solve the inequality: 3(2x + 5) - 2(3 - x) < 4x - 7

Answer: x > 11/8

8. Find c: 2(c + 3) - 3(2c - 5) ≥ 7 - 4c

Answer: c ≤ 17/7

9. Solve for y: (1/4)(16 - 2y) + 5 > (1/3)(9 - y)

Answer: y < 17/5

10. Determine d: 3(2d + 7) ≤ 5(2 - d) + 3

Answer: d ≤ 19/7

Simultaneous Linear Equations

1. Solve the system:

2x + 3y = 11

4x - y = 5

Answer: x = 2, y = 3

2. Solve the system:

3a + 2b = 14

5a - 4b = 6

Answer: a = 4, b = 2

3. Solve the system:

5c - 2d = 8

3c + d = 9

Answer: c = 2, d = 3

4. Solve the system:

4p + 7q = 35

2p - 3q = 1

Answer: p = 3, q = 4

5. Solve the system:

3x - 2y = 5

2x + y = 8

Answer: x = 3, y = 2
