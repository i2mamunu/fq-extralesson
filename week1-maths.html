Linear Equations Questions

Simple Questions:

  1. Solve for \(x\): \(2x + 5 = 11\)
    Answer: \(x = 3\)
  2. Find the value of \(y\): \(4(y - 3) = 20\)
    Answer: \(y = 8\)
  3. Determine \(a\): \(\frac{3}{4}a - 5 = 7\)
    Answer: \(a = 16\)
  4. Solve the equation: \(\frac{2x}{3} + 4 = 10\)
    Answer: \(x = 9\)

Harder Questions:

  1. Solve for \(x\): \(3x - 2(4x + 5) = 7\)
    Answer: \(x = -\frac{17}{11}\)
  2. Find the value of \(y\): \(\frac{2y + 7}{3} = \frac{5y - 1}{2}\)
    Answer: \(y = \frac{29}{7}\)
  3. Determine \(a\): \(2(a + 3) - 5(a - 1) = 12\)
    Answer: \(a = -\frac{7}{3}\)
  4. Solve the equation: \(5 - \frac{2x + 3}{4} = \frac{x - 1}{2}\)
    Answer: \(x = 18\)
  5. Solve for \(x\): \(2(3x - 1) + 4 = 5(2x + 2)\)
    Answer: \(x = \frac{6}{7}\)
  6. Find the value of \(m\): \(\frac{2(m - 1)}{3} + 4 = \frac{m + 5}{2}\)
    Answer: \(m = -\frac{7}{4}\)
